# Paraflow Legal Website

A professional legal website built with React and Tailwind CSS, following the comprehensive style guide for a clean, minimalist light theme with deep professional navy blue and golden yellow accents.

## Features

- **React 18** with modern hooks and components
- **Tailwind CSS** for styling following the professional legal style guide
- **React Router** for navigation
- **Responsive design** with professional legal aesthetic
- **Iconify icons** for consistent iconography

## Pages

- **Home** - Hero section, practice areas, why choose us, latest insights, and CTA
- **About Us** - Firm story, values, team members, and credentials
- **Services** - Detailed practice areas with key services
- **Blog** - Legal insights and updates with search and filtering
- **Contact** - Contact form, office information, and consultation booking

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview production build:

```bash
npm run preview
```

## Style Guide Compliance

All components follow the professional legal style guide:

- Primary colors: Deep navy blue (#1B3A5F) with lighter/darker variants
- Accent colors: Golden yellow (#D4A745) used sparingly
- Typography: System font stack with proper sizing and weights
- Spacing: Consistent spacing system (Tight, Compact, Standard, Relaxed, Section)
- Borders: Subtle fine-line borders (#E8ECF0)
- Layout: Vertical layout with top header navigation
